# Kairo

Kairo es un proyecto de finanzas, donde fijamos dos estructuras o mejor dicho dos formas de uso
El primero es para aquellas personas que no se rigen por un sueldo fijo o por un gasto fijo
Es decir, no tienen un ingreso estático de un trabajo 9-5, o tampoco tienen un gasto fijo como la renta o servicios de streamming pero en cantidad

El segundo es para aquelas personas que sí se rigen por el sueldo fijo o gasto fijo, añdiendo la parte de contabilidad a los cálculos necesarios para un mejor análisis

Las tecnologías del proyecto, el STACK : - Next.js + TypeScript - Node + Express + TypeScript - PostgreSQL - Prisma - JWT (importante)
